# Velvet Crypt - Gothic Fantasy Cocktail Bar Website

A luxury immersive cocktail bar website featuring dark fantasy aesthetics with elegant rose-purple tones, inspired by gothic architecture and mystical ambiance.

## Design System

### Color Palette
- **Deep Black** (#0a0a0f) - Primary background
- **Dark Purple** (#1a0d2e) - Secondary background
- **Velvet Violet** (#2d1b4e) - Tertiary background
- **Dark Rose** (#c77d94) - Primary accent (from logo)
- **Gothic Pink** (#d896ad) - Secondary accent
- **Soft Silver** (#c8c8d4) - Text color

### Typography
- **Display/Headings**: Cinzel (Gothic serif)
- **Body**: Inter (Modern sans-serif)
- **Editorial**: Cormorant Garamond (Classic serif)

### Features
- 8 fully responsive pages with React Router
- Smooth animations using Motion/Framer Motion
- Gothic architectural inspired design
- Custom logo integration throughout
- Masonry gallery layout
- Interactive reservation form
- Mobile-friendly navigation

## Pages
1. **Home** - Hero section with logo, featured elixirs, experience preview
2. **About** - Brand story and philosophy
3. **Elixirs** - Full cocktail menu with 8 signature drinks
4. **Experience** - Atmosphere details, VIP lounge, dress code
5. **Events** - Weekly and annual event calendar
6. **Gallery** - Masonry photo grid
7. **Reservations** - Booking form with validation
8. **Contact** - Location, hours, FAQ

## Tech Stack
- React 18
- TypeScript
- Tailwind CSS v4
- React Router v7
- Motion (Framer Motion)
- Lucide Icons
- React Responsive Masonry

## Development
```bash
pnpm install
pnpm dev
```

## Design Notes
The website captures the essence of "elegant mystery" with a cinematic, immersive feel perfect for a gothic fantasy cocktail bar. The rose-pink tones from the logo create a sophisticated, luxury aesthetic that avoids typical vampire red clichés.
